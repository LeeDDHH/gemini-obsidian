# Stable Diffusion Web UI 일본어 현지화 프로젝트

**원문:** https://github.com/L4Ph/stable-diffusion-webui-localization-ja_JP

**요약:**
이 GitHub 저장소는 AUTOMATIC1111 버전의 Stable Diffusion web UI를 일본어로 현지화하는 프로젝트입니다. Crowdin을 통해 번역에 참여할 수 있으며, 2개 국어 동시 표시를 지원하는 Bilingual Localization 확장 기능과의 병용을 권장합니다. 설치 방법과 사용 방법, 그리고 프로젝트의 디렉토리 구조에 대한 설명이 포함되어 있습니다.
