# 10 open source tools that feel illegal...

## TL;DR
Kali Linux에 기본 탑재된 10가지 오픈소스 윤리적 해킹 도구를 소개하는 영상. NMAP(네트워크 매핑), Wireshark(패킷 분석), Metasploit(해킹 프레임워크), Aircrack-ng(Wi-Fi 해킹), Hashcat(비밀번호 크래킹), Skipfish(웹 취약점 스캔), Foremost(포렌식 데이터 복구), SQLmap(SQL 인젝션), hping3(DoS 공격), Social Engineering Toolkit(피싱)을 다루며, 모든 도구는 동의 없이 사용 시 불법임을 강조한다.

## 요약
영상은 컴퓨터 전문가를 사용자·프로그래머·해커 세 부류로 나누며 시작한다. 사용자는 해커에게 사회공학적 기법으로 당하고, 프로그래머도 백도어를 열어두면 수십억 달러 손실을 초래할 수 있다고 경고한다. 이어서 Kali Linux 환경에서 사용 가능한 10가지 도구를 시연한다:

1. **NMAP** - IP 대역에 패킷을 보내 열린 포트와 OS를 감지하는 네트워크 매핑 도구. `-A` 옵션으로 적극적 스캔과 트레이서 루트까지 가능하다.
2. **Wireshark** - 수백 가지 프로토콜의 네트워크 트래픽을 실시간 캡처·분석하는 GUI 도구.
3. **Metasploit** - "AK-47이 달린 스위스 군용 칼"로 비유되는 가장 강력한 해킹 프레임워크. EternalBlue 취약점으로 Windows 7 대상 리버스 셸 시연.
4. **Aircrack-ng** - Wi-Fi 패킷을 캡처하고 WPA 키를 크래킹. HTTPS의 중요성을 강조.
5. **Hashcat** - rockyou.txt 같은 사전 파일로 해시된 비밀번호를 크래킹. MD5는 수초 만에, bcrypt는 며칠 걸릴 수 있음.
6. **Skipfish** - 웹사이트를 재귀 크롤링하며 XSS, SQL 인젝션 등 취약점 스캔.
7. **Foremost** - 파일 카빙 기반 포렌식 복구 도구. 빠른 포맷된 디스크에서도 데이터 복원 가능.
8. **SQLmap** - 데이터베이스 스키마를 매핑하고 SQL 인젝션 공격을 자동화.
9. **hping3** - flood 옵션으로 DoS/DDoS 공격 시연. 봇넷과 결합 시 DDoS로 확대.
10. **Social Engineering Toolkit** - 이메일, QR코드, SMS, 웹사이트 복제 등으로 피싱 공격 생성.

## 핵심 주장/근거/예시
- **주장**: 이 도구들은 사이버 보안 학습과 침투 테스트에 필수적이나, 동의 없는 사용은 국제법 위반으로 징역형까지 가능하다.
- **근거**: 각 도구의 실제 시연을 통해 위력을 보여주며, Metasploit 사용 시 "사이버 보안 학습 기회를 놓치게 된다"는 경고로 기초 학습의 중요성도 강조.
- **예시**: Windows 7의 EternalBlue 취약점으로 리버스 셸 획득, rockyou.txt로 MD5 해시 수초 내 크래킹, 웹사이트 복제를 통한 피싱 시연.

## 실무 적용 포인트
- 자체 서비스의 보안 점검 시 NMAP과 Skipfish로 네트워크/웹 취약점 사전 진단 가능
- 비밀번호 정책 수립 시 Hashcat의 크래킹 속도를 참고하여 bcrypt 등 강력한 해싱 알고리즘 필수 적용
- HTTPS 미적용 시 Aircrack-ng 같은 도구로 평문 데이터 탈취가 가능하므로, 모든 폼에 HTTPS 강제 필요
- 개발자도 기본적인 침투 테스트 지식을 갖추어 백도어나 SQL 인젝션 취약점을 사전에 예방해야 함

## 검증/추가 확인 질문

3. 기업 환경에서 합법적인 침투 테스트를 수행하기 위한 절차와 인증(CEH, OSCP 등)은?
4. bcrypt 외에 현재 권장되는 비밀번호 해싱 알고리즘(Argon2 등)의 비교?
