# Stable Diffusion Web UI 모델(Checkpoint) 사용 방법

**원본 URL**: https://soroban.highreso.jp/article/article-040

**요약**:
이 문서는 Stable Diffusion Web UI(AUTOMATIC1111 및 Forge 버전)에서 모델(Checkpoint)을 사용하는 방법을 설명한다. Stable Diffusion은 텍스트를 기반으로 다양한 스타일의 이미지를 생성하는 무료 AI 도구이다. 모델은 이미지 생성 시 스타일과 분위기에 영향을 미치며, Hugging Face나 Civitai에서 다운로드할 수 있다. 다운로드한 모델 파일은 Stable Diffusion WebUI의 특정 폴더에 저장해야 하며, Google Colab에서도 사용할 수 있다. 또한, 모델 사용 전 라이선스 및 상업적 이용 가능 여부를 확인하는 것이 중요하다.
