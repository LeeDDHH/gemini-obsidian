# Stable Diffusion 이미지 기형 방지 방법

## 요약
이 글은 Stable Diffusion으로 생성된 이미지에서 손이나 신체 왜곡과 같은 기형을 방지하는 방법을 설명합니다. 주요 방법으로는 512, 768, 1024픽셀과 같은 최적의 이미지 크기 사용, 손과 신체 부위에 대한 특정 부정 프롬프트 적용, LoRA 또는 Embedding 활용을 제안합니다. 이러한 방법들이 매번 완벽한 결과를 보장하지는 않지만, 사용하지 않을 때보다 이미지 품질을 크게 향상시킨다고 강조합니다.
