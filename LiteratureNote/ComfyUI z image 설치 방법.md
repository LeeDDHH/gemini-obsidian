# ComfyUI z image 설치 방법

[Source: https://www.reddit.com/r/comfyui/comments/1pc0aau/i_am_new_to_comfyui_how_can_i_install_z_image/?tl=ko]

Reddit 게시물은 초보자를 위해 ComfyUI에 "z 이미지"를 설치하는 방법을 묻고 있습니다. 한 사용자가 설치 방법을 안내합니다. 먼저, GitHub 링크(comfyanonymous.github.io/ComfyUI_examples/z_image/)에서 파일 3개를 다운로드하여 올바른 폴더에 넣습니다. 그 다음, 여성 이미지를 ComfyUI로 드래그하여 워크플로우를 로드하고, 프롬프트를 작성한 후 실행 버튼을 클릭합니다. GPU나 RAM이 부족한 사용자는 작은 모델 크기를 사용하는 것이 좋습니다. 아직 ComfyUI가 설치되지 않은 Windows 및 Nvidia GPU 사용자에게는 GitHub(github.com/Tavris1/ComfyUI-Easy-Install/tree/Windows)의 원클릭 설치 프로그램이 권장됩니다. 또한, ComfyUI에 익숙하지 않은 사람들을 위해 YouTube 채널(@pixaroma)에서 튜토리얼을 시청하는 것을 추천합니다.
