# TL;DR
Theo는 Anthropic이 많은 Claude Code 사용자들에게 사용량 대시보드에서 "claim"을 클릭하면 50달러의 무료 크레딧을 제공하고 있다고 알립니다.

# 요약
Theo - t3.gg는 트윗을 통해 Anthropic이 자사의 AI 코딩 도구인 Claude Code 사용자들에게 50달러 상당의 무료 크레딧을 제공하고 있다는 소식을 전했습니다. 사용자들은 Claude Code 사용량 대시보드에 접속하여 "claim" 버튼을 클릭함으로써 이 크레딧을 받을 수 있습니다. 이 프로모션은 Claude Code의 사용을 장려하고, 잠재적으로 더 많은 개발자들이 AI 기반 코딩 지원을 경험하게 하기 위한 Anthropic의 전략으로 해석됩니다. 이는 AI 개발 도구 시장에서의 경쟁이 심화되고 있음을 보여주는 사례이기도 합니다.
