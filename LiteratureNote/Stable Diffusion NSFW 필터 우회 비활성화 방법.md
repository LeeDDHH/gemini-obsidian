# Stable Diffusion NSFW 필터 우회/비활성화 방법

**원본 URL**: https://mania.romptn.com/article/stable-diffusion/45/

**요약**:
Stable Diffusion에서 NSFW 필터를 우회하거나 비활성화하는 방법은 Negative prompt에 'nsfw', 'nude', 'revealing clothes', 'on bed'와 같은 단어를 추가하여 효과를 강조하는 것입니다. 이러한 방법을 통해 NSFW 콘텐츠 생성을 방지할 수 있지만, AI 이미지 생성의 무작위성 때문에 100% 확실하지는 않습니다. NSFW는 "Not Safe For Work"의 약자로, 직장에서 보기에 부적절한 콘텐츠를 의미하며, 필터에 걸리면 이미지가 검게 처리되거나 볼 수 없게 됩니다.
