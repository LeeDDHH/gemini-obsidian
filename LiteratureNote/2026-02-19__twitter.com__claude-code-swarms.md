# TL;DR
iwashi는 Claude Code의 숨겨진 기능인 'Swarms'를 소개하며, 이를 팀 리더가 코드 작성 대신 계획, 위임, 통합을 담당하는 것에 비유합니다.

# 요약
iwashi / Yoshimasa Iwase는 Claude Code에 대한 흥미로운 게시물에서 "Swarms"라는 새로운 숨겨진 기능을 언급합니다. 이 기능은 마치 서브팀과 대화하는 듯한 느낌을 주며, 팀 리더가 코드를 직접 작성하기보다는 계획을 세우고, 작업을 위임하며, 최종 결과를 통합하는 역할에 비유됩니다. 이는 AI 개발에서 팀 리더십의 역할 변화를 시사하며, Claude Code가 단순히 코드를 생성하는 도구를 넘어 프로젝트 관리 및 협업 방식에 영향을 미칠 수 있음을 보여줍니다.
