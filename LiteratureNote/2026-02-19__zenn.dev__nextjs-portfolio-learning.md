# 프론트엔드 아무것도 모르는 상태에서 Next.js 포트폴리오 사이트를 만들기까지 공부한 내용 메모

**출처**: [Zenn](https://zenn.dev/zerebom/scraps/23376e6f791279)
**저자**: zerebom
**작성일**: 2024-02-25 / 최종 업데이트: 2024-08-10
**도메인**: zenn.dev

## TL;DR

- AI/UX 양쪽을 다룰 수 있는 인재의 필요성을 인식하여 프론트엔드 학습 시작
- React → Next.js(App Router) 순서로 학습, ChatGPT 활용으로 속도 향상
- 정적 사이트 제작이 이전보다 훨씬 간단해진 현 환경에서 직접 포트폴리오 사이트 제작 성공

## 요약

### 학습 배경

AI와 UX를 모두 다룰 수 있는 인재의 필요성 인식 + ChatGPT 등 AI 도구 발전으로 프론트엔드 학습 진입 장벽이 낮아진 점을 고려하여 현대적 프론트엔드 기술 습득을 결심.

### 주요 학습 영역

- **React 기초**: 튜토리얼 및 핵심 개념
- **Web 기초**: HTML, CSS, JavaScript
- **스타일링**: Tailwind CSS
- **프레임워크**: Next.js (App Router 기반)
- **디자인**: Figma
- **방법론**: 소프트웨어 설계 및 테스트

### 산출물

- React + Next.js로 포트폴리오 웹사이트 구현
- RSS를 통한 블로그 통합 기능 개발

### 핵심 통찰

- "정적 사이트 제작이 이전보다 훨씬 간단해졌음"
- 다른 개발자의 코드를 참고하기 쉬워 학습 속도 빠름
- 기술 트렌드 변화 속도가 빠른 점이 지속적 과제

## 핵심 포인트

- ML/AI 연구자가 프론트엔드를 학습하는 경로 사례
- App Router 기반 Next.js 학습이 현재 권장 경로
- ChatGPT를 멘토로 활용한 학습법이 효과적

## 내 적용 아이디어

- 비(非)프론트엔드 배경을 가진 사람의 프론트엔드 학습 여정 참고
- Next.js App Router 기반 학습 리소스 모음으로 활용
