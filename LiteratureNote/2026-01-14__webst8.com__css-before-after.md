## TL;DR

CSS 의사 요소 `::before`와 `::after`는 HTML 마크업을 늘리지 않고 CSS만으로 자식 요소를 추가하는 기법이다. `content` 속성이 필수이며, 검색 엔진에 색인되지 않으므로 장식용으로만 사용해야 한다. 제목 부제, NEW 뱃지, 커스텀 리스트 마커, 장식 테두리, 말풍선 등 다양한 UI 패턴에 활용할 수 있다.

## 요약

### ::before / ::after 기본 개념

- 선택한 요소의 콘텐츠 **앞(::before)** 또는 **뒤(::after)** 에 의사 요소를 삽입
- `content` 속성은 빈 문자열(`""`)이라도 반드시 선언해야 동작
- 기본적으로 인라인 요소로 동작하며, `display: block`으로 변환 가능

### 주요 활용 패턴

1. **제목 부제**: 일본어 제목 아래 영문 텍스트를 `::after`로 삽입
2. **NEW 뱃지**: `position: absolute`와 조합하여 알림 아이콘 생성
3. **커스텀 리스트 마커**: 기본 불릿을 이미지로 교체
4. **장식 테두리**: 일본식 괄호(`「」`) 등으로 텍스트 감싸기
5. **말풍선(speech bubble)**: 삼각형 꼬리 포인터를 의사 요소로 구현

### 중요 주의사항

- 검색 엔진은 의사 요소의 텍스트를 일반 HTML 콘텐츠로 색인하지 않음
- SEO에 중요한 정보는 반드시 실제 HTML 요소로 작성할 것
- Font Awesome 아이콘도 의사 요소를 통해 삽입 가능

## 핵심 포인트

- `::before`/`::after`는 장식과 레이아웃 보조에 최적화된 기법
- `content` 속성 없이는 의사 요소가 렌더링되지 않음
- `position` 속성과 조합하면 복잡한 UI 패턴도 마크업 없이 구현 가능
