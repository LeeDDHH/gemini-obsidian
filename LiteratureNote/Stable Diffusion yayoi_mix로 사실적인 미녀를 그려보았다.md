# Stable Diffusion 'yayoi_mix'로 사실적인 미녀를 그려보았다

**원본 URL**: https://climbtown.jp/use_yayoi_mix/

**요약**:
이 글은 Stable Diffusion의 "yayoi_mix" 모델을 사용하여 사실적인 미녀 이미지를 생성하는 방법을 다룹니다. 저자는 CIVITAI의 특정 이미지를 재현하려 했으나, 얼굴 디테일과 기모노 패턴에서 차이를 보였습니다. 이는 `txt2img` 설정만으로는 부족하며, 원본 이미지가 `img2img`나 업스케일링과 같은 추가 기능을 사용했을 가능성을 시사합니다. 저자는 향후 `txt2img` 외의 다른 Stable Diffusion 기능들을 탐구할 계획입니다.
