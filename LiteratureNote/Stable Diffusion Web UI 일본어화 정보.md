# Stable Diffusion Web UI 일본어화 정보

**원문:** https://wikiwiki.jp/sd_toshiaki/%E6%97%A5%E6%9C%AC%E8%AA%9E%E5%8C%96

**요약:**
이 문서는 Stable Diffusion Web UI의 일본어화에 대한 정보를 제공합니다. "stable-diffusion-webui-localization-ja_JP" 확장 기능을 통해 거의 완벽하게 일본어화가 가능하며, Crowdin에서 일본어화 팀이 활동하고 있습니다. 2개 국어 동시 표시 기능도 지원하지만, 영어 UI가 사용자 수가 많아 질문이나 검색 시 더 효율적일 수 있습니다. 일본어 UI를 직접 만들거나 사용자 정의하는 방법도 설명되어 있습니다.
