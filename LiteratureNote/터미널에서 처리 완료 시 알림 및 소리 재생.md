# 터미널에서 처리 완료 시 알림 및 소리 재생

## Summary
이 문서는 Mac 터미널에서 시간이 오래 걸리는 작업이 완료되었을 때 알림이나 소리를 재생하는 방법을 설명합니다. `afplay` 명령어를 사용하여 시스템 사운드를 재생하거나, `tput bel`로 비프음을 낼 수 있습니다. 또한, `osascript` 명령어를 활용하여 "Process is done!"과 같은 메시지를 포함한 알림을 표시할 수 있습니다. 이러한 기능들을 `.zshrc` 파일에 별칭(alias)으로 등록하여 편리하게 사용할 수 있으며, `&&` 연산자를 사용하여 명령 실행 후 소리와 알림을 동시에 발생시키는 방법도 제시합니다. 이로써 사용자는 터미널 작업 완료를 효율적으로 인지할 수 있습니다.이 문서는 Mac 터미널에서 시간이 오래 걸리는 작업이 완료되었을 때 알림이나 소리를 재생하는 방법을 설명합니다. `afplay` 명령어를 사용하여 시스템 사운드를 재생하거나, `tput bel`로 비프음을 낼 수 있습니다. 또한, `osascript` 명령어를 활용하여 "Process is done!"과 같은 메시지를 포함한 알림을 표시할 수 있습니다. 이러한 기능들을 `.zshrc` 파일에 별칭(alias)으로 등록하여 편리하게 사용할 수 있으며, `&&` 연산자를 사용하여 명령 실행 후 소리와 알림을 동시에 발생시키는 방법도 제시합니다. 이로써 사용자는 터미널 작업 완료를 효율적으로 인지할 수 있습니다.[1][1]
