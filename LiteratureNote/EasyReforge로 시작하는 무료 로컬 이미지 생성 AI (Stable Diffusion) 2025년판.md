# EasyReforge로 시작하는 무료 로컬 이미지 생성 AI (Stable Diffusion) 2025년판

**원본 URL**: https://zenn.dev/km100/articles/8a65774aeb580f

**요약**:
이 문서는 EasyReforge를 사용하여 로컬 환경에서 Stable Diffusion 이미지 생성 AI를 설치하고 사용하는 방법을 설명합니다. EasyReforge는 Stable Diffusion, reForge, EasyReforge의 세 가지 도구의 관계를 설명하며, 특히 초보자도 쉽게 사용할 수 있도록 필요한 모델, 확장 기능, 설치 프로그램을 하나로 묶은 올인원 패키지입니다. 이 도구는 원클릭 설치, 일본어 지원, 고품질 이미지 생성을 위한 템플릿 및 사전 설정, Hires. fix 및 ADetailer와 같은 고품질화 도구의 사전 설치 등의 주요 특징을 가지고 있습니다. 또한, EasyReforge의 시스템 요구 사항, 설치 절차, 첫 이미지 생성 방법, 프롬프트 작성법, 강조 구문 활용, LoRA(Low-Rank Adaptation)를 통한 모델 사용자 정의, 그리고 TIPO(Text to Image with text Presampling for prompt Optimization) 확장 기능을 활용한 프롬프트 자동 생성 및 확장 기능에 대해서도 자세히 설명합니다.
