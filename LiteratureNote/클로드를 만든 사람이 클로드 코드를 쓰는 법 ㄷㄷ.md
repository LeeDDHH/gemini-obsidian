# 클로드를 만든 사람이 클로드 코드를 쓰는 법 ㄷㄷ
https://www.youtube.com/watch?v=Mf0Nkwstkng

## **요약**

  

클로드 코드(Claude Code) 제작자가 제안한 사용 팁은 **여러 개의 클로드 세션을 동시에(병렬로) 돌려 작업 처리량을 끌어올리는 방식**입니다.

구체적으로는 **1~5번처럼 세션에 번호를 붙여** 각각을 독립적인 작업자처럼 운용하고, **시스템 알람(알림)**을 이용해 “응답 확인 → 다음 지시 입력” 루프를 **여러 세션에서 동시에** 반복합니다.

  

여기서 더 나아가, **웹사이트에서 5~10개의 클로드 세션을 동시에 실행**해 더 큰 병렬화를 만들고, **Claude iOS 앱으로 세션을 먼저 시작해 두었다가** 나중에 시간을 내서 결과를 확인·추가 지시하는 식으로 **시작과 확인을 분리**하는 운영도 언급됩니다.

---

## **핵심 포인트**

- **세션 번호화(1~5)**: 각 세션을 서로 다른 역할/작업으로 분리해서 혼선 없이 병렬 처리
    
- **알람 기반 루프**: 알람이 울리면 해당 세션 결과 확인 → 다음 입력 → 다시 알람 설정
    
- **웹에서 확장(5~10 세션)**: 병렬 처리 규모를 더 키워 처리량 극대화
    
- **모바일로 시작, 나중에 확인**: iOS에서 세션을 ‘발사’해두고, 나중에 데스크탑/웹에서 회수(확인·정리)
