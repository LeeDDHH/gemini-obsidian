프론트엔드 지식이 전혀 없는 상태에서 Next.js를 이용해 포트폴리오 웹사이트를 구축하기까지의 학습 과정을 기록한 글이다. AI와 UX를 겸비한 인재의 중요성을 인지하고, AI 기술 발전이 프론트엔드 학습을 용이하게 할 것이라는 생각으로 모던 프론트엔드 학습을 시작했다.

학습 과정은 React 튜토리얼을 통해 기본기를 다지고, 시마부(しまぶー)와 같은 온라인 강의를 통해 광범위한 기술 스택을 익혔다. 다른 개발자들의 Next.js 포트폴리오 구현 사례를 분석하고, GitHub Copilot의 도움을 받아 코드를 이해하고 활용했다. Tailwind CSS, Figma, CSS Grid & Flex, shadcn/ui 등 다양한 프론트엔드 기술을 학습했으며, Next.js 공식 문서를 통해 개발 환경을 구축하고 App Router를 도입했다.

회고를 통해 프론트엔드 기술 트렌드 파악의 중요성과 정적 사이트 구축의 용이성, 그리고 프론트엔드 개발자의 고민(상태 관리, SSR, SSG 등)에 대한 이해를 높였다. 또한, UI 디자인을 넘어선 소프트웨어 설계의 중요성을 깨닫고 객체 지향 및 클린 아키텍처와 같은 설계 방법론을 탐구했다. 테스트 코드 작성, 개인 프로젝트의 효율적인 개발 방안(Supabase 활용 등)에 대해서도 학습했다.
