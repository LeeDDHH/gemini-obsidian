# Next.js 프로덕션 환경 사용법 및 BFF 활용

**Original URL**: https://kaminashi-developer.hatenablog.jp/entry/2026/01/06/just-use-next

**Summary**:
이 글은 Next.js를 프로덕션 환경에서 사용하는 방법에 대해 다루며, 특히 Next.js를 BFF(Backend For Frontend)로 활용하는 것을 강조한다. 저자는 Next.js 서버를 신뢰할 수 없는 환경으로 간주하고, 민감한 정보를 저장하지 않으며, 최소한의 권한을 부여하여 보안 위험을 줄이는 것을 제안한다. 또한, Next.js의 RSC(React Server Component)가 개발 경험과 BFF로서의 적합성 때문에 Next.js를 선택하는 중요한 이유라고 설명한다.
