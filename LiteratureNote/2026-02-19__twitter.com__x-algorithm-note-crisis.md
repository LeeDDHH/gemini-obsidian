# TL;DR
tanu는 최근 X(트위터) 알고리즘 변화로 인해 사용자들이 Note 플랫폼 링크 대신 X의 자체 게시 기능으로 수익을 창출하는 경향이 생겨, Note 플랫폼이 사용자 유입 감소로 위기에 처할 수 있다고 지적합니다.

# 요약
tanu는 최근 X(구 트위터)의 알고리즘 변화가 콘텐츠 크리에이터들의 행동에 미치는 영향에 대해 언급합니다. 일부 사용자들이 'note'와 같은 외부 플랫폼의 링크를 공유하는 대신, X의 자체 게시 기능을 활용하여 수익을 얻는 것이 더 유리하다는 것을 깨닫고 있다고 합니다. 이러한 변화는 단순히 사용자 점유율 경쟁을 넘어, X로부터 'note' 플랫폼으로의 트래픽 유입이 크게 감소할 수 있음을 의미하며, 이는 'note' 플랫폼에게 심각한 위협이 될 수 있다고 분석합니다. 이는 플랫폼 알고리즘이 콘텐츠 전략 및 수익 모델에 미치는 중요한 영향을 보여주는 사례입니다.

# 원문 (blockquote)
```
Xの最近のアルゴリズムで「noteのリンクを貼る代わりに記事機能を使ってXで稼いだ方が稼げる」という事に気づいた人がちらほら現れてて、本格的にnoteやばい気がする。単にユーザーの取り合いならまだしも、Xからの流入が激減してるはず。
```

# 내 적용 아이디어 (실무)
*   콘텐츠 플랫폼 전략 수립 시, 외부 채널의 알고리즘 변화가 자사 플랫폼으로의 트래픽 유입에 미치는 영향을 주기적으로 분석하고 대비책을 마련합니다.
*   자체 플랫폼 내에서 콘텐츠 크리에이터가 수익을 창출할 수 있는 다양한 방법을 모색하고, 외부 플랫폼과의 연동 전략을 재검토하여 경쟁력을 강화합니다.
*   특정 소셜 미디어 플랫폼에 대한 의존도를 줄이기 위해 다양한 채널을 통한 콘텐츠 배포 및 사용자 확보 전략을 다각화합니다.
