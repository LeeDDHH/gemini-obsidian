# Chris' Corner: 최신 CSS 동향

**Original URL**: https://blog.codepen.io/2026/01/05/chris-corner-freshly-fallen-css/?ref=dailydev

**Summary**:
이 블로그 게시물은 CSS의 최신 개발 사항들을 다룹니다. 특히, "masonry" 레이아웃이 "grid-lanes"라는 새로운 이름으로 WebKit에 도입되었으며, `text-grow` 및 `text-shrink`와 같은 새로운 텍스트 크기 조정 속성이 프로토타입 단계에 있습니다. 또한, 스크롤 트리거 애니메이션과 사용자 정의 가능한 `<select>` 요소의 발전된 기능에 대해서도 소개합니다.
