# Stable Diffusion Web UI LoRA 사용법 (추가 학습 모델)

**원본 URL**: https://soroban.highreso.jp/article/article-041

**요약**:
이 기사는 Stable Diffusion Web UI(AUTOMATIC1111 및 Forge 버전)에서 추가 학습 모델인 LoRA를 사용하는 방법을 설명합니다. LoRA는 Stable Diffusion 모델(Checkpoint)에 특정 스타일이나 요소를 추가하는 데 사용되는 차등 파일입니다. Hugging Face나 Civitai와 같은 플랫폼에서 LoRA 파일을 다운로드하는 방법과 Windows, Mac, GPUSOROBAN 환경 및 Google Colab에 LoRA 파일을 설정하는 방법을 안내합니다. LoRA를 사용하면 사용자가 원하는 이미지에 더 가까운 결과물을 생성할 수 있습니다.
