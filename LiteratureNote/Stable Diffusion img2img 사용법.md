# Stable Diffusion img2img 사용법

## 요약
이 문서는 Stable Diffusion의 img2img 기능을 사용하여 이미지를 기반으로 새로운 이미지를 생성하는 방법을 설명한다. 텍스트 프롬프트만으로는 어려운 특정 포즈나 스타일을 구현할 때 유용하며, 원본 이미지의 특징을 유지하면서 애니메이션 스타일로 변환하는 예시를 제공한다. Denoising strength를 조절하여 원본 이미지와의 유사성을 조정하고, Resize mode의 "Resize and fill" 옵션을 통해 이미지 크기를 유연하게 변경하는 방법을 다룬다.
