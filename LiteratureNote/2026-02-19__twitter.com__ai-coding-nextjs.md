# TL;DR
VoxelKei는 Next.js와 React에 대해 전혀 모르는 상태에서 Claude Code와 Antigravity를 사용하여 단 한 줄의 코드도 직접 작성하지 않고 웹 개발을 성공했다고 공유하며, "Agentic Coding"의 놀라움을 강조합니다.

# 요약
VoxelKei는 웹 개발에 대한 지식이 전무한 상태에서(Next.js와 React에 대한 기본 개념조차 모르는 수준) Claude Code와 Antigravity라는 도구를 활용하여 프로젝트를 완성했다고 밝혔습니다. 그는 이 과정에서 단 한 줄의 코드도 직접 작성하지 않았음을 강조하며, 이러한 "Agentic Coding" 방식의 엄청난 잠재력과 효율성에 대해 놀라움을 표현했습니다. 이 사례는 AI 기반 코딩 도구가 비전문가도 복잡한 웹 애플리케이션 개발에 참여할 수 있게 하는 새로운 가능성을 제시함을 보여줍니다.
