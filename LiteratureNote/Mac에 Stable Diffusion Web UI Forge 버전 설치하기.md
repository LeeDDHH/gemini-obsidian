# Mac에 Stable Diffusion Web UI Forge 버전 설치하기

**원본 URL**: https://soroban.highreso.jp/article/article-089

**요약**:
이 문서는 Stable Diffusion Web UI Forge 버전을 Mac 로컬 PC에 설치하는 방법을 안내합니다. Forge 버전은 AUTOMATIC1111 버전과 유사하지만, VRAM 사용량을 절약하고 이미지 생성 속도를 향상시키는 장점이 있습니다. 특히 VRAM이 6GB 또는 8GB인 GPU에서 효과가 크며, Homebrew를 이용한 설치 과정과 사용법을 상세히 설명합니다.
