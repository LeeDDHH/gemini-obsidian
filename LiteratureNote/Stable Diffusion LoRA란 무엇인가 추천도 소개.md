# Stable Diffusion 'LoRA'란 무엇인가? 추천도 소개

**원본 URL**: https://aismiley.co.jp/ai_news/what-is-stable-diffusion-lora/

**요약**:
Stable Diffusion의 "LoRA"는 모델의 일부 매개변수를 미세 조정하는 기술로, Stable Diffusion에 대규모 추가 학습 없이 모델을 맞춤 설정할 수 있게 해줍니다. LoRA는 "Low-Rank-Adaptation"의 약자로, 적은 리소스로 효율적인 학습이 가능하다는 의미를 담고 있습니다.

LoRA를 사용하면 다음과 같은 작업이 가능합니다.
*   **화풍 조정**: "애니메이션 스타일" 또는 "사실적인 스타일"과 같은 화풍을 조정할 수 있습니다.
*   **고품질 이미지 생성**: 간단한 프롬프트로도 고품질 이미지를 생성할 수 있습니다.
*   **복잡한 디자인 재현**: 프롬프트만으로는 생성하기 어려운 복잡한 디자인도 정밀하게 재현할 수 있습니다.
*   **인물 및 캐릭터 지정**: 특정 인물이나 캐릭터를 지정하여 이미지를 생성할 수 있습니다.
*   **의상, 헤어스타일, 배경 지정**: 의상, 헤어스타일, 배경 등을 상세하게 지정하여 생성할 수 있습니다.
*   **인물 포즈 지정**: 인물의 포즈를 지정하여 더욱 구체적인 이미지를 생성할 수 있습니다.

LoRA 모델은 주로 Civitai와 Hugging Face에서 배포됩니다. Civitai는 Stable Diffusion 사용자 간의 정보 교환 커뮤니티 플랫폼이며, Hugging Face는 AI 모델과 데이터셋을 공유하고 활용할 수 있는 플랫폼입니다.

LoRA는 일반적으로 로컬 환경에서 사용되며, Stable Diffusion Web UI에서 사용하는 방법은 다음과 같습니다.
1.  `stable-diffusion-webui` 폴더를 클릭합니다.
2.  `models` 폴더를 클릭합니다.
3.  `Lora` 폴더를 클릭하고 다운로드한 LoRA 파일을 저장합니다.
4.  `stable-diffusion-webui`를 실행합니다.
5.  화면 오른쪽의 빨간색 태그 아이콘을 클릭합니다.
6.  화면에 왼쪽에 표시되는 `LoRA` 탭을 클릭합니다.
7.  사용하려는 LoRA를 클릭합니다.

기사에서는 `costume cute kimono`, `Pixel Portrait`, `Eye – LoRa`, `Doll style`, `blindbox`, `LineAniRedmond`, `flat2`, `:3`, `Anime Girls Holding Signs` 등 다양한 추천 LoRA 모델을 소개하고 있습니다.
