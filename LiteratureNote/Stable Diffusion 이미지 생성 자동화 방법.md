# Stable Diffusion 이미지 생성 자동화 방법

**원문:** https://www.reddit.com/r/StableDiffusion/comments/153t8rt/is_it_possible_to_automate_image_generation_with/?tl=ja

**요약:**
이 레딧 게시물은 시드를 동일하게 유지하면서 약간 다른 프롬프트로 이미지 생성을 자동화하는 방법에 대해 묻습니다. 사용자들은 X/Y/Z 플롯, `txt/list` 옵션, 또는 와일드카드를 사용한 동적 프롬프트를 제안합니다. 한 사용자는 처음에 와일드카드가 "무작위"하지만 반복 가능한 순서로 이미지를 생성하는 문제에 직면했지만, 나중에 "조합 옵션"을 찾아 해결했습니다. 토론에서는 자동화를 위해 Python으로 API를 사용하는 것도 언급됩니다.
