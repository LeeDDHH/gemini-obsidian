# Mac에 Stable Diffusion Web UI 설치하기

**원본 URL**: https://soroban.highreso.jp/article/article-037

**요약**:
이 문서는 Mac 로컬 PC에 Stable Diffusion Web UI(AUTOMATIC1111 버전)를 설치하는 방법을 설명합니다. Stable Diffusion은 텍스트를 이미지로 변환하는 무료 AI이며, Web UI는 브라우저를 통해 쉽게 사용할 수 있는 애플리케이션입니다. 설치 과정에는 Homebrew 설치, Stable Diffusion Web UI 저장소 복제, `webui-macos-env.sh` 파일 편집, 그리고 Web UI 실행이 포함됩니다. 또한, Forge 버전과의 차이점과 GPU 메모리 부족 시 PICSOROBAN과 같은 클라우드 서비스 사용을 권장합니다.
