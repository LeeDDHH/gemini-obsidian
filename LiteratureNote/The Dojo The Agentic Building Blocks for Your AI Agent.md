# The Dojo The Agentic Building Blocks for Your AI Agent

## **CopilotKit Dojo 요약 (Generative UI 중심)**

  

CopilotKit Dojo는 **CopilotKit 팀이 만든 인터랙티브 플레이그라운드**로, “에이전틱(Agentic) 경험”과 **Generative UI**를 만들 때 어디서부터 시작해야 하는지에 대한 **멘탈 모델 + 핵심 빌딩 블록**을 제공합니다.

대화형 챗을 기본으로 두고, 에이전트가 **툴 호출 / 상태 변화 / 사용자 승인** 등을 통해 UI를 직접 “구동”하는 패턴들을 단계적으로 보여줍니다.

---

## **Dojo가 소개하는 핵심 빌딩 블록**

  

### **1) Agentic Chat (기본 토대)**

- LLM(또는 LangGraph 에이전트)와 연결된 **라이브 채팅 UI**
    
- 스트리밍 응답과 대화 경험을 “기본 제공”
    
- 여기서 더 나아가 **프론트엔드 툴**에도 접근 가능
    
    - 예: “배경색 바꿔줘” → 앱 UI가 실제로 변경됨
        
    
- 즉, _커스텀 Generative UI를 얹기 전_ 출발점
    

---

### **2) Backend Tool Rendering (Generative UI 시작점)**

- 에이전트가 **백엔드에서 툴 호출**하면,
    
- 그 결과를 **커스텀 React 컴포넌트로 채팅 중간에 렌더링**
    
- 툴 실행 중엔 로딩 상태를 보여주고 완료되면 결과로 교체 가능
    
    - 예: 텍스트 대신 “날씨 카드”가 실시간으로 표시
        
    
- 컴포넌트가 API와 연결되어 있어 **데이터도 동적**
    

---

### **3) Human-in-the-loop (사용자 승인/거절로 에이전트 제어)**

- 툴 렌더링과 유사하지만 **respond callback**이 추가됨
    
- 에이전트가 어떤 작업을 진행하다가 **중간에 멈추고 사용자 입력을 기다림**
    
- 사용자가 “Approve/Reject”하면 그 결과가 에이전트로 전달되어 다음 단계 진행
    
- DB 삭제, 결제/거래 처리 등 **실제 영향이 큰 작업에 필수적인 안전장치**
    

---

### **4) Tool-based Generative UI (툴 정의 방향을 반대로)**

- 기존: 백엔드가 툴 정의 → 에이전트가 호출
    
- 여기서는: **프론트엔드(브라우저)에서 툴을 정의** → 에이전트가 호출
    
- 프론트 툴은 **DOM 상태 / 브라우저 API / React 앱 내부 상태**에 접근 가능
    
- CopilotKit의 state sync로 에이전트가 이런 툴들을 “발견”하고 활용
    

---

### **5) Agentic Generative UI (툴 호출 단위가 아니라 “상태” 기반 렌더링)**

- “툴 호출마다 컴포넌트를 렌더링”하는 방식이 아니라
    
- 에이전트의 **진행 상태(state)가 업데이트**될 때마다 UI가 바뀌는 방식
    
- render 함수가 **현재 state**를 받아서 단계별 UI를 구성
    
    - 예: 계획을 세우는 동안 단계 체크리스트가 계속 업데이트되어 표시됨
        
    

---

### **6) Shared State (채팅 밖의 앱 UI까지 양방향 연결)**

- 에이전트 ↔ React 프론트엔드 사이에 **양방향 상태 브릿지**를 구성
    
- 앱 UI에서 상태를 바꾸면 에이전트가 그 변화를 “인지”
    
- 예: 카드 제목을 “carrot cake”로 바꾸고
    
    에이전트에게 “우리가 뭘 굽고 있지?”라고 물으면 상태를 기반으로 답함
    
- 내부적으로 useAgent 훅이 state / setState를 제공해서 동작
    

---

## **결론**

  

CopilotKit Dojo는 “에이전트가 UI를 구동한다”는 관점에서,

**대화 → 툴 렌더링 → 사용자 승인 → 프론트 툴 → 상태 기반 UI → 앱 전체 공유 상태**까지의 패턴을 한 번에 체험하게 해주는 학습장입니다.

영상은 dojo.agui.com에서 직접 실습해보라고 안내하며, Discord 커뮤니티 참여도 권장합니다.
