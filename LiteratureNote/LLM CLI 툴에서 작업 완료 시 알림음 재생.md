# LLM CLI 툴에서 작업 완료 시 알림음 재생

## Summary
이 글은 LLM CLI 도구에서 작업 완료 시 알림 소리를 재생하는 방법을 설명합니다. 모든 OS에서 간단한 비프음을 내는 방법과 Mac에서는 `afplay`, Ubuntu에서는 `mplayer`를 사용하여 사용자 지정 오디오 파일을 재생하는 방법을 제시합니다. VSCode 터미널 설정 변경에 대한 주의사항도 언급합니다. 핵심은 `GEMINI.md` 파일에 작업 완료 시 소리 명령을 실행하는 코드를 추가하는 것입니다.
