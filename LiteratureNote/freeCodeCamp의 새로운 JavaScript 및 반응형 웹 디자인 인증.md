# freeCodeCamp의 새로운 JavaScript 및 반응형 웹 디자인 인증

**Original URL**: https://app.daily.dev/posts/freecodecamp-launches-new-javascript-and-responsive-web-design-certifications-lbxjhx8ho

**Summary**:
freeCodeCamp는 JavaScript, Python 및 반응형 웹 디자인을 다루는 세 가지 새로운 인증 프로그램을 시작했습니다. 각 인증은 수백 시간의 대화형 콘텐츠, 5개의 실습 프로젝트, 그리고 부정 행위 방지 조치가 포함된 50문제 시험으로 구성됩니다. 이 프로그램은 DOM 조작, 비동기 JavaScript, Python 데이터 구조 및 알고리즘, CSS Flexbox 및 Grid를 사용한 반응형 디자인과 같은 핵심 개념을 다룹니다. 모든 인증은 무료이며 영구적으로 유효하며 웹캠이나 신분증 확인이 필요하지 않습니다.
